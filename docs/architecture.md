# CestaSmart - Arquitectura técnica inicial

## Frontend
- Next.js + React + TypeScript.
- UI responsive para uso móvil y desktop.

## Backend
- Node.js + TypeScript.
- API REST para autenticación, catálogo, precios y carrito.

## Datos
- PostgreSQL para datos transaccionales.
- Redis para caché y sesiones.

## Ingesta
- Conectores API oficial y scraping legalmente viable.
- Jobs en cola para refresco de precios y disponibilidad.

## Módulos iniciales
1. Auth/Users.
2. Products/Catalog.
3. Retailers/Zones.
4. Prices/Offers.
5. Cart/Optimization.
6. Ticket/Checkout summary.
7. Admin.
