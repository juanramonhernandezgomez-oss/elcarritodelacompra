# CestaSmart - Plan de desarrollo por sprints

## Sprint 1
- Setup monorepo y CI básica.
- Registro/login.
- Modelo de datos inicial.
- Selector de zona.
- Búsqueda de producto y carrito base.

## Sprint 2
- Primeros conectores de supermercados.
- Matching exacto por EAN + marca + formato.
- Precios separados (base/fidelización/promo).

## Sprint 3
- Optimizador de cesta (1 tienda / mínimo coste / equilibrio).
- Ticket final completo con ahorro.
- Dashboard básico de histórico de precios.
