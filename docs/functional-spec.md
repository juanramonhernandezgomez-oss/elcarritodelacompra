# CestaSmart - Documento funcional (MVP + Fase 2)

## MVP

### Objetivo
Comparar precios de productos de supermercados en España, permitiendo construir una cesta optimizada según diferentes estrategias de ahorro.

### Funcionalidades clave
- Registro/login obligatorio.
- Geolocalización por GPS, CP, municipio y barrio.
- Filtro de supermercados por disponibilidad en zona (online/presencial).
- Comparación por producto exacto (EAN + marca + formato).
- Visualización separada de precio base, fidelización y promoción.
- Cálculo de coste final incluyendo envío, pedido mínimo y recargos.
- Ticket final con desglose por supermercado y ahorro estimado.
- Listas recurrentes semanales/mensuales.
- Panel admin para supermercados, mapeo de productos y usuarios.
- Histórico de precios y dashboard de evolución.

## Fase 2
- Perfiles de hogar (personas, alergias, preferencias).
- Recomendación inteligente de sustitutos.
- Alertas de bajada de precio.
